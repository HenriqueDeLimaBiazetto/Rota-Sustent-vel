# Rota-Sustentável

Rota Sustentável é um site interativo

O projeto combina informações educativas, visualização de dados e simulações.

## Funcionalidades

* Painel de Diagnóstico
Apresenta informações sobre infraestrutura rural, cadeia de frio e perdas de alimentos durante o transporte.
Exibe indicadores e métricas relacionados ao desperdício alimentar.
* Simulador de Rotas
Permite selecionar pontos no mapa da América Latina.
Calcula a distância aproximada entre origem e destino.
Estima o percentual de perda de carga com base no percurso realizado.
* Jogo Educativo
O jogador controla um caminhão de transporte de alimentos.
É necessário evitar obstáculos e coletar cargas para manter a eficiência da entrega.
Possui sistema de pontuação, dificuldade ajustável e ranking.
* ♿ Recursos de Acessibilidade
- Ajuste do tamanho da fonte.
- Alto contraste.
- Escala de cinza.
- Contraste negativo.
- Modo claro.
- Fonte de maior legibilidade.
- Destaque visual para elementos interativos.
Objetivo

O principal objetivo do projeto é demonstrar como problemas de infraestrutura, transporte e armazenamento podem contribuir para o desperdício de alimentos, incentivando a reflexão sobre soluções mais sustentáveis para a cadeia logística.
